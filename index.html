Comunicação Client x Server

Entender a comunicação cliente-servidor é essencial para todo aquele que tem contato com a tecnologia, pois está presente em tudo que utilizamos que tenha vínculo com a internet, principalmente com a navegação web. É possível entender que a troca de dados ocorre por dois lados, sempre ambos enviando e recebendo, a didática do curso facilitou a compreensão. (Voltar ao Início)

Hardware x Software

Ao decorrer de algumas aulas, foi possível entender que o software depende do hardware para que as aplicações sejam executadas, assim como o hardware depende de do software, sendo assim fica claro que toda programação está em software e consequentemente o hardware a executa, se assim estiver destinado.
